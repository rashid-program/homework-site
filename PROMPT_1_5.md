# ПРОМПТ ДЛЯ ОБНОВЛЕНИЯ ЛОГИКИ ЗАДАНИЙ №1–5 (ОГЭ)

Нужно реализовать новую логику обработки заданий №1–5. Эти задания представляют собой единый составной блок и всегда используются вместе.

---

## 1. Общая логика заданий №1–5

Задания №1–5:

- являются единым комплектом  
- не разделяются  
- всегда загружаются, хранятся и выводятся вместе  
- относятся к одному прототипу одновременно  

Каждый прототип содержит **сразу все 5 заданий**.

---

## 2. Типы (прототипы) заданий №1–5

Существует 7 прототипов:

01 — Дороги  
02 — Квартиры  
03 — Участки  
04 — Печи  
05 — Бумага  
06 — Шины  
07 — Мобильный интернет  

---

## 3. Логика хранения в базе данных

В базе данных:

- прототипы хранятся как и раньше(числами а не словами)  
- текстовые названия НЕ хранятся  
- один прототип = один комплект из 5 заданий  

---

## 4. Логика отображения на сайте

При выводе пользователю числа заменяются на названия:

- 1 → Дороги  
- 2 → Квартиры  
- 3 → Участки  
- 4 → Печи  
- 5 → Бумага  
- 6 → Шины  
- 7 → Мобильный интернет  

Это применяется в:

- `index.html`  
- странице базы заданий  
- странице генерации варианта  
- PDF-файле варианта  

---

## 5. Особая логика отображения (ВАЖНО)

Для заданий №1–5 **НЕ используется** старая механика сайта.

### ❌ Как было раньше:
Прототип → список заданий → предпросмотр примера

### ✅ Как нужно теперь:
Пользователь выбирает только **ТИП (прототип)** без вложенных списков.

### Пример интерфейса:

**Задания №1–5**

- ☐ Дороги  
- ☐ Квартиры  
- ☐ Участки  
- ☐ Печи  
- ☐ Бумага  
- ☐ Шины  
- ☐ Мобильный интернет  

Без раскрытия.  
Без предпросмотра.  
Без вложенности.

Пользователь выбирает **количество прототипов**, а не задания внутри них.

## 9. Важно учесть

Это особый тип заданий.

Он отличается тем, что:

- имеет составную структуру  
- не требует предпросмотра  
- упрощает выбор для пользователя  
- работает по принципу  
  **«один тип = один комплект»**

Необходимо реализовать чистую архитектуру, чтобы:

- код был масштабируемым  
- можно было добавлять новые типы  
- логика не дублировалась
